# customer-churn-analysis
Analysis of customer churn data to identify factors influencing customer retention using Python and machine learning

## Overview
This project analyzes customer churn data to identify key factors driving customer attrition and builds a machine learning model to predict churn.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
Customer churn dataset used during my remote internship at Saiket Systems.

## Key Insights
- Month-to-month contracts have higher churn rates
- Customers with higher monthly charges are more likely to churn
- Tenure is a strong indicator of customer retention

## Conclusion
The analysis provides actionable insights to improve customer retention strategies.
